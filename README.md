
# togemory

## 🎞️ 소개

투게모리는 여행을 기록하고 함께 추억을 공유하는 웹사이트입니다.

Together와 Memory를 합친 뜻으로 함께한 여행을 사진과 글로 남기고 같이 추억 할 수 있었으면 좋겠다는 취지로 시작하게 되었습니다. 

## 📌추가할 기능

* 여행 기록을 게시글 형태로 작성
* 게시글 열람시 조회수 반영 
* 사진첨부 및 여행 위치 공유
* 댓글기능 추가
